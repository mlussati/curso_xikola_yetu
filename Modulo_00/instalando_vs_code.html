Instalando Visual Studio Code

O Visual Studio Code é um editor de código criado pela Microsoft e que tem uma grande adoção pelas comunidades de diversas linguagens
e tecnologias. Nós iremos usar ela neste curso inicialmente para quem tiver o interesse de executar o curso via notebooks pois ela disponibiliza
uma extensão para isso.

Requisitos para instalar

  1. Python instalado
  2. Configurações da máquina recomendável ter: pelo menos Intel Core i3, Dual Core, com 8GB de Ram e no mínimo 128GB de disco (não que seja necessário
    é apenas para questão de performance)
  3. Sistema Operacional SEMPRE ATUALIZADO, e em caso de uso do Windows, utilizar sempre a versão 10 ou superior.

Instalando no Windows

A instalação no Windows é relativamente simples, basta acessar o site oficial do Visual Studio Code e fazer o Download da
sua última versão, clicando no botão "Download for Windows".


Em seguida, execute o instalador baixado e siga as instruções na tela, não esqueça de marcar a opção "Add to Path" para adicionar o Visual Studio Code nas variáveis de ambiente.


Finalizada a instalação, feche todos os terminais abertos e abra um novo (Power Shell ou Windows Terminal) e digite o comando abaixo.

code --version
