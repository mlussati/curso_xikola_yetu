Instalando Python

Instale algumas ferramentas que serão importante para que você começar a fazer o curso:

Nós iremos disponibilizar duas formas de fazer o curso a primeira será você fazer o curso via local no seu computador,
a outra opção seria usar o o binder jupyter que eu já irei ilustrar como utilizar ela, mas é a visão mais recomendável,
para estes modulos iniciais.

Local, instale:

Abaixo uma descrição detalhada de como instalar os programas:

  1. Python
  2. Sublime text
  3. Vs Code

1. Python

Se você usa o Windows acesse:

https://www.python.org/downloads/windows/

Passo 1:


E baixe a última versão disponiblizada no site Python 3 (exemplo autalmente a versão mais recente é Python 3.9.1)



(Para quem usa o Linux ou Mac o Python já vem instalado)

Passo 2: Faça o download do instalador executável do Windows e clique duas vezes nele para executar e iniciar
de instalação do Python como é mostrado abaixo:

Atenção!! quando instalei o python na minha máquina eu usava a versão 3.7.4 então na vossa instalação é suposto aparecer
a versão do python que vocês baixaram

O processo de instalação é muito simples:

  1. Marque a opção "Add Python to PATH"
  2. Clique em "Install Now"

A tela abaixo será mostrada. Aguarde enquanto o instalador completa o processo de instalação.


E depois a conclusão será mostrada o seguinte:

Para verificar se a instalação do Python foi bem-sucedida, pesquise no menu iniciar por "cmd" e clique duas vezes para abri-lo.

Digite o comando:

python --version

Este comando retornará a versão do python que está instalada em sua máquina.
